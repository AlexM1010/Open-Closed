# Open-Closed

A monorepo containing personal life management projects built on Structural Optimism principles.

## Projects

### [life-launcher](./life-launcher)
Android launcher with integrated life management features, energy tracking, and Google sync.

- **Platform**: Android (Kotlin)
- **Type**: Launcher app with home screen widget
- **Repository**: https://github.com/AlexM1010/life-launcher

### [life-manager](./life-manager)
Full-stack web application for life domain balance, task planning, and health tracking.

- **Platform**: Web (React + Node.js)
- **Type**: Local-first web app with SQLite
- **Repository**: https://github.com/AlexM1010/life-manager

## Philosophy

Both projects implement the Structural Optimism philosophy: integration across life domains creates wellbeing. The system treats relationship and health tasks as genuinely important, not optional.

## Setup

```bash
# Clone with submodules
git clone --recurse-submodules https://github.com/AlexM1010/Open-Closed.git

# Or if already cloned, initialize submodules
git submodule update --init --recursive
```

## Development

Each project has its own README with specific setup instructions. Navigate to the project directory and follow its documentation.
